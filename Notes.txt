PYTHON NOTES
-------------------
variable : A variable in Python is a named container used to store data that can be changed during program execution.
--------------------------------------------
scripts : A set of instructions
--------------------------------------------
constants: The value that is stored in variable is known as constants OR A constant in Python is a value that is meant to stay the same and not change while the program runs. 
--------------------------------------------
assignment operator: An assignment operator in Python is used to give a value to a variable (like =). "=" is also known as assignment operator 
--------------------------------------------
Arithmetic operator: In Python, arithmetic operators are symbols used to perform basic math operations like addition, subtraction, multiplication, and division.
--------------------------------------------
Operands and Operators: Operands are the values you use (like numbers or variables) 
Operators (operations) are the symbols that perform actions on them
for example in a+b
a and b are operands and + is a operator
--------------------------------------------
Data Types: In Python, data types are the types of values a variable can store (like numbers, text, or true/false).
--------------------------------------------
data types are there in python:

🔹 Basic Data Types
int → whole numbers (e.g., 10, -5)
float → decimal numbers (e.g., 3.14)
str → text (e.g., "hello")
bool → True or False
🔹 Collection Data Types
list → ordered, changeable collection (e.g., [1, 2, 3])
tuple → ordered, but not changeable (e.g., (1, 2, 3))
set → unordered, unique items (e.g., {1, 2, 3})
dict → key-value pairs (e.g., {"name": "Muskan"})
--------------------------------------------
What is type casting?
type casting means converting one data type into another.
--------------------------------------------

what are conditional statements?
Conditional statements are used to make decisions based on conditions (true or false). They help your program choose what to do next based on a condition.

if/else condition:
if/else condition is used to choose between two actions. If condition is true → do one thing, otherwise → do another thing.

what is arrray and list?
List → a collection of items that can be changed and store multiple type data
Array → similar to a list but usually stores same-type data and needs libraries like NumPy

append() → adds an item at the end of the list
insert() → adds an item at a specific position
----------------------------------------------------------------------------------------------------------------------
what is iteration of list?

Iteration means repeating steps over items one by one. Iteration means going through each item in a list one by one.

What are Library?
libraries are collections of pre-written code that help you do tasks easily. Libraries are ready-made tools that you can use instead of writing code from scratch.
----------------------------------------------------------------------------------------------------------------------
Few useful python libraries for devops:

os → used to interact with the operating system (files, folders, paths)
👉 create/delete folders, check files, get current directory

shutil → used for file operations like copy, move, delete
👉 copy files, move files, delete folders

system (os.system) → used to run system commands (like terminal commands)
👉 like running mkdir, dir, ls, etc.

Functions in Python

In Python, a function is a block of code that performs a specific task and can be reused whenever needed.

🔹 What is a Function?
A function is used to group some code together so you can run it many times without rewriting it.

Why Do We Use Functions?
Functions are useful because they:
Avoid repeating code (write once, use many times)
Make programs shorter and cleaner
Make code easier to understand
Help divide big programs into small parts
